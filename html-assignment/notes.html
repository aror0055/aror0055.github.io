Image Tag:
Image tag <img> is used to display an image on a webpage. This tag a singular tag and does not require a closing tag. The tag consists of 2 pieces of information which are as follows:
  1. Source i.e. src: this is the relative or the absolute path to the image which needs to be displayed using the same img tag.
  2. Alternate text i.e. alt: This text provides a description of the image which needs to be displayed using the same img tag in a case where either the link is broken or the image does not exist in the specifies path. This description helps provide a description for the image in a search engine and screen reader.
Anchor Tag:
The anchor tag <a> is used to link html web pages, pictures, documents and other resources to the page from which the link is being accessed from in order to pass information or or to go to another source of information. This tag requires a closing tag </a>. The tag consists of 2 pieces of information which are as follows:
  1. Hyper Reference i.e. href: This is piece of information is the hyperlink to the piece of information that needs to be linked to the web page.
  2. Display Text: This text provides a Text that can be used as a button to open the hyperlink that is being stored in this text.
Form Tag:
The form tag <form> is used to create a fillable form. This form consists of text fields, radio buttons, checkboxes, etc. This tag requires a closing tag </form>. The tag consists multiple attributes in order to be used in a desirable way but there are 2 main pieces of information that are required for this tag to function which are as follows:
  1. Action i.e. behavior: This is piece of information decides the type of input that needs to be entered/selected in a field. It is of various types, for example, text, number, email, etc.
  2. Method: This piece of information defines how the information will be sent. There are two types of methods which are as follows:
    • get: it is used when there is no sensitive information in the information that is being sent to it's destination. For example, media content on an open source media library like YouTube.
    • post: it is used when there is sensitive information in the information that is being sent to it's destination. For example, credit card details.
Table Tag:
Table tag <table> is used is used to create a table using standard html code. This tag requires a closing tag </table>. This tag can be used to display information in an organized manner on a webpage. This tag consists of columns and rows which are alloted spaces on a webpage to display the provided information.
Navigation:
Navigation Tag <table> is used is used to navigate between different sections of a web page or to navigate between multiple web pages in the same project/website. This tag requires a closing tag </nav>.
